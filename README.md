# [f78's website](https://www.f78.be)

The third version of my website, which isn't quite finished yet, and may therefore contain some issues. This time around
I'm using [Astro](https://astro.build), with the final website being deployed to Cloudflare Workers.

*Third time's the charm, right?*

## Improvements over the old Jekyll site

- An improved home page
- Better styling... again (based on [Modern Normalize](https://github.com/sindresorhus/modern-normalize))
- A work-in-progress animated starry background using WebGL
- Many modernizations thanks to Astro, such as responsive images
